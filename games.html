Unreal Engine 4 (2021-2022)

Shadow Racer

Shadow Racer is a top-down racing game where the player needs to defeat enemies by using different skills, explore the world solving different puzzles and unlocking power ups to defeat darkness.

Note: the project is not finished yet, will be published on summer 2022

More information about the game

I am working on Shadow Racer this year which is my senior year at DigiPen. We are using Unreal Engine 4 to build the game.

Team: 6 programmers

My contributions untill now

Role: AI & Gameplay programmer.

  • I implemented the Enemy AI
  • Built the main level of the game (level 2).
  • Helped building the tutorial level
  • Designed and built the game cinematics
  • Helped on game design

Custom Engine (2020-2021)

Seavilizations

Seavilizations is a 3D multiplayer boat-combat game in which all the players have to fight against each other to obtain a certain number of chalices so that they can escape from the map to win.

  • Download

More information about the game

I worked on Seavilizations during my junior year at DigiPen. The game was built on a custom C++ engine that we did from scratch.

Team: 8 programmers

My contributions

Role: Designer and UI & Network programmer.

  • I designed and implemented the whole networking system of the game. This system makes possible to play up to 4 players in LAN
  • Helped on building the object component system
  • I implemented all the menus of the game
  • I worked on the design team of the game

Custom Engine (2018-2019)

Back To Life

Back To Life is a 2D Puzzle-Platformer game where death is a way to help you reach your goal through discovery, trial and error.

  • Download

More information about the game

I worked on Back To Life during my sophomore year at DigiPen. The game was built on a custom C++ engine that we did from scratch.

Team: 5 artists and 5 programmers

My contributions

Role: Engine & Gameplay programmer and Co-Producer of the programmers team.

I implemented the following features to our custom 2D engine:

  • Engine Editor
  • Engine Serializer
  • Object Manager used by the serializer to build the levels when loading a file
  • Game object component system

I also contributed to the design and built some of the levels
